About Image Secrets

Image secrets lets you hide and unhide messages within images using steganography.

How to use

Drag any image into the image view on the left side. To hide a message, enter the message into the text field and press 'Hide Message'. A new image in the format filename.hidden.filetype will be created in the same directory as the original image with the message encoded withn it.

Likewise, drag an image with a hidden message into the application and press unhide message to reveal the hidden message.

Enjoy using ImageSecrets to play with the idea of steganography.
